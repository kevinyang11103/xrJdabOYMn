# PHP+Mysql游戏信息管理系统 java383

## 项目概述

本项目是一款基于PHP和Mysql开发的综合性游戏信息管理系统。系统提供用户登录、注册、游戏信息增删改查、分页浏览以及图片上传等基本功能。

## 技术栈

- 后端语言：PHP
- 数据库：Mysql
- 集成环境：phpstudy、Wampserver、Xampp

## 功能模块

### 用户模块

- 登录与注册
- 查看游戏详情
- 浏览所有游戏
- 修改个人资料
- 上传头像
- 收藏与取消收藏游戏

### 管理员模块

- 用户管理（删除操作）
- 游戏信息管理
- 发布新游戏信息
- 查看游戏类型数量占比报告

## 系统角色

- 普通用户
- 管理员

## 运行环境

- PHP版本 5.6 或 7

## 目录结构

```plaintext
- / (根目录)
  - /public (公共资源目录)
  - /uploads (图片上传目录)
  - /application (应用程序目录)
    - /controller (控制器目录)
    - /model (模型目录)
    - /view (视图目录)
  - /system (系统核心目录)
  - /database (数据库文件目录)
```

## 部署步骤

1. 准备PHP和Mysql运行环境
2. 解压项目文件到服务器指定目录
3. 导入数据库文件至Mysql数据库
4. 配置服务器，确保访问权限正确
5. 访问网站进行测试

- 系统界面友好，操作简单
